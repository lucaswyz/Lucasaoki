Títulos servem para definir o tema de um site. Existem seus níveis de títulos

Podemos destacar trechos de texto para enfatizar uma ideia

  • Uma lista não ordenada não se importa
  • UL significa "unordered list"
  • LI significa List Item
  1. Uma lista ordenada é uma lista que possue algum tipo de ordem numérica
  2. Seu funcionamento é exatamente com uma UL
  3. Ol significa "Ordered List"
